==============================
Production Tool - EXE Version
==============================

This EXE allows you to configure and program vibration monitoring devices during production.

-----------------------
🧩 Requirements
-----------------------
- Windows 10 or 11 (64-bit)
- USB connection to device
- No installation required (portable EXE)

-----------------------
🚀 How to Run
-----------------------
1. Extract the entire ZIP/exe package to a known folder (e.g., `C:\ProductionTool`)
2. Double-click the executable:
   > ProductionToolV1.exe

3. Follow the on-screen interface to:
   - Load Master configuration
   - Setup Device settings
   - Send commands to the remote hardware

-----------------------
⚠️ Notes & Precautions
-----------------------
- Do **NOT** delete or rename any files inside the folder.
- Do **NOT** run from a temporary or read-only location.
- Avoid using paths with special characters (e.g., spaces, symbols).
